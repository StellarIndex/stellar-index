# Inventory Artifacts

Generated by:

```sh
./docs/audit-2026-04-29/inventory/generate.sh
```

Files:

- `repo-snapshot.md`: commit, worktree, counts, scope notes.
- `area-counts.md`: grouped counts by major repo area.
- `file-coverage.tsv`: one row per tracked file for per-file audit status.

`file-coverage.tsv` is the audit's non-negotiable coverage ledger.

Every tracked file must end with a terminal status:

- `done`
- `excluded`
- `blocked`

No `todo` rows may remain at closure.
