# Journey Evidence

Supporting evidence per-journey. Each subfolder or file collects
the artifacts that don't fit cleanly under a single evidence
ledger.

Journey traces themselves live at
[../../journeys-traces/](../../journeys-traces/), one file per
journey.

Use this directory for:

- captured payloads (XDR fixture excerpts, vendor response samples)
- raw SQL query results
- network capture excerpts (sanitised)
- screenshots if relevant

Naming: `<journey-id>-<artifact-description>.<ext>`. Each artifact
referenced from the corresponding journey trace.
